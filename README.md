# 🎮 EveryDIE
> Unreal Engine 기반 서바이벌 호러 게임 프로젝트  
> 현재 개발 단계입니다. 주요 맵들을 구성하는 중입니다.
> 현재 완성 및 업데이트 단계입니다. 완성된 게임을 기반으로 확장, 버그 수정 등을 진행합니다.
> Windows 폴더에 있는 EveryDIE.exe 파일을 실행하여 게임에 진입합니다. 종료는 Alt + f4로 종료해주세요.

## 🧭 프로젝트 개요
이 프로젝트는 서바이벌 호러 게임으로,  
플레이어가 괴물로 뒤덮인 캠퍼스에서 살아남는 것을 목표로 합니다.  

현재는 **프로젝트 완성 및 버그 수정 단계**입니다. 버그를 제보 받고 있습니다.

## 📖 세계관 & 스토리 (초기 버전 요약)
> 이제 막 서울과학기술대학교에 입학한 26학번 새내기, 다음날 개강을 맞아 설레는 마음을 품고 기숙사에서 잠이 든다.  
> 다음날 아침, 기숙사를 나서는 길에 괴물들이 학생들을 무자비하게 공격하는 모습을 본 주인공은 기숙사로 다시 도망을 친다.
> 가지고 있는 유일한 전자기기인 핸드폰으로 뉴스를 보자 전국적으로 괴물이 출현했다는 소식을 듣는다.
> 전국적인 소식만 전하는 뉴스는 뒤로하고 캠퍼스의 상황을 알기 위해 에브리타임에 들어가는 주인공. 과연 에브리타임으로 여러 정보를 얻고 생존자끼리 힘을 합쳐 이 사태에서 살아남을 수 있을까?

## 📖 세계관 & 스토리 (변경 버전 요약)
> 캠퍼스를 좀비들이 점령하게 되고 많은 사람들이 죽어 나가고 있다. 이런 상황에서 주인공은 최대한 물건들을 찾아 살아남으려고 한다.
> 가지고 있는 유일한 전자기기인 핸드폰으로 에브리다이를 확인하자, 전국적으로 괴물이 출현했다는 소식을 듣고 캠퍼스가 아수라장이 된 것을 알게 된다.
> 전국적인 소식만 전하는 뉴스는 뒤로하고 캠퍼스의 상황을 알기 위해 에브리다이에 들어가는 주인공. 과연 에브리다이으로 여러 정보를 얻고 생존자끼리 힘을 합쳐 이 사태에서 살아남을 수 있을까? (* 여기서 에브리다이는 대학 생활 앱 에브리타임을 모방한 프로그램)

## 🎮 Controls
| 키 | 기능 |
|----|------|
| W A S D | 이동 |
| Spacebar | 점프 |
| Shift | 달리기(스태미나 소모 기능 있음) |
| E | 아이템 / 상자 상호작용 / NPC 대화하기 |
| 1,2,(3) | 인벤토리 무기 퀵슬롯 (1 : 맨손, 2 : 권총, 3 : 라이플(라이플은 이스터에그로 327호에 숨겨놓았습니다.)) |
| I | 인벤토리 / 에브리다이 휴대폰 열기 |
| Q + 클릭 | 인벤토리에서 아이템 버리기
| 인벤토리 아이템 클릭 | 캔 고기나 에너지 드링크 사용하기

## 🌟 Features
- 기본 이동 및 FPS 전투 시스템
- 아이템을 보관하는 인벤토리 시스템 (코드 + UI) 구현
- 좀비 AI 추적 시스템 사용
- 체력, 스태미나, 인벤토리 / 휴대폰 UI, 조준점 UI 사용

- 스마트폰 UI 기반 정보 시스템 (에브리다이)

## 🧪 Tested Environment
- Windows 11
- NVIDIA RTX 4070 Laptop 사용
- RAM 64GB 사용

## 📦 Build Instructions
1. Unreal Engine 5.5.4 설치
2. 프로젝트 열기
3. File → Package Project → Windows 선택
4. Build 완료 후 Shipping 폴더에서 실행 파일 확인

## 🐛 Known Issues
- 좀비들이 가끔 바닥이나 벽에 끼는 문제 발생

## 게임 공략
- 초반에는 이동 속도가 빠른 좀비에 주의하며, 물자를 확보하세요.
- 에브리다이로 캠퍼스 상황을 확인하고 생존자 위치를 파악하세요.
- 인벤토리에서 음식/음료를 적절히 사용해 체력과 스태미나를 관리하세요.
- 라이플은 특정 위치(327호)에 숨겨져 있으며, 초반에는 권총만 사용 가능합니다.
- 분기 선택에 따라 엔딩이 달라지므로 다양한 시도를 권장합니다.
