# 🎮 Project Title (가제)
> Unreal Engine 기반 서바이벌 호러 게임 프로젝트  
> 현재 구상 단계입니다. 아이디어를 발전시키며 초기 컨셉을 정리 중입니다.

---

## 🧭 프로젝트 개요
이 프로젝트는 서바이벌 호러 게임으로,  
플레이어가 괴물로 뒤덮인 캠퍼스에서 살아남는 것을 목표로 합니다.  

현재는 **스토리, 세계관, 시스템 구상 중**이며, 이후 언리얼 엔진을 기반으로 구현할 예정입니다.

---

- [ ] 기본 프로토타입 제작 (Unreal Engine)

## 📖 세계관 & 스토리 (요약)
> 이제 막 서울과학기술대학교에 입학한 26학번 새내기, 다음날 개강을 맞아 설레는 마음을 품고 기숙사에서 잠이 든다.  
> 다음날 아침, 기숙사를 나서는 길에 괴물들이 학생들을 무자비하게 공격하는 모습을 본 주인공은 기숙사로 다시 도망을 친다.
> 가지고 있는 유일한 전자기기인 핸드폰으로 뉴스를 보자 전국적으로 괴물이 출현했다는 소식을 듣는다.
> 전국적인 소식만 전하는 뉴스는 뒤로하고 캠퍼스의 상황을 알기 위해 에브리타임에 들어가는 주인공. 과연 에브리타임으로 여러 정보를 얻고 생존자끼리 힘을 합쳐 이 사태에서 살아남을 수 있을까?
